# Querying Github API and Local Search

Using React, fetch public repositories and order by fork count descending. Includes local multi term partial match search capability.

### GraphQL API

- Check out `graphql` branch to see the app query against Github's GraphQL API

### REST API

- Check out `master` branch to see the app query against Github's REST API
